== Description ==

<strong>NOTICE! This is the last version! We have published a new plugin called <a href="https://wordpress.org/plugins/modal-links/">Modal Links</a> that includes this plugin's functionality and many more... We are developing on that for now and on. <br />Thank you</strong>

This plugin opens a post/page in modal window using the post/page permalink in shortcode.<br />
Install it and activate it to see it in action.<br />
Go to Settings->Modal Permalinks and check the options.

== Installation ==

1. Upload the folder "modal-permalinks" to the "/wp-content/plugins/" directory or install it through WordPress directly.
2. Activate the "Modal Permalinks" plugin through the 'Plugins' menu in WordPress.
3. Go to Settings->Modal Permalinks and check the options.
